# DAX40 Financial Analysis with Tableau

## Overview

This repository contains a series of financial analysis projects completed using Tableau and DAX40 company financial data.

The project focuses on applying financial statement analysis, ratio analysis, trend analysis, and peer group benchmarking to evaluate company performance and support investment decisions.

## Project Objectives

* Analyze the financial health of DAX40 companies.
* Apply liquidity, profitability, solvency, and activity ratios.
* Develop interactive Tableau dashboards.
* Interpret financial performance and trends.
* Formulate evidence based investment recommendations.

## Skills Demonstrated

* Financial Statement Analysis
* Ratio Analysis
* Trend Analysis
* Peer Group Benchmarking
* Tableau Dashboard Development
* Data Visualization
* Investment Evaluation
* Business Research
* Financial Reporting

## Project Structure

### Task 1: Single Year Analysis

Financial ratio analysis of Adidas AG for the most recent reporting period.

### Task 2: Trend Analysis

Multi year financial trend analysis of Adidas AG to identify changes in performance over time.

### Task 3: Peer Group Analysis

Comparative analysis of BMW against Mercedes Benz Group AG and Volkswagen AG using industry benchmarking techniques.

* Tableau
