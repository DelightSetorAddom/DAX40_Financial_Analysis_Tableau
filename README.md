# DAX40 Financial Analysis with Tableau

## Overview

This repository contains a series of financial analysis projects completed during the Financial Analysis course at the University of Würzburg.

Using Tableau and DAX40 company financial data, the projects evaluate corporate performance through financial statement analysis, trend analysis, and peer group benchmarking. The analyses were used to support investment recommendations based on financial evidence.

## Skills Demonstrated

* Financial Statement Analysis
* Financial Ratio Analysis
* Trend Analysis
* Peer Group Benchmarking
* Tableau Dashboard Development
* Data Visualization
* Investment Evaluation
* Business Research
* Team Collaboration
* Financial Reporting

## Project Structure

### Task 1: Single Year Analysis

Financial analysis of Adidas AG using liquidity, profitability, solvency, and activity ratios to evaluate financial performance and investment attractiveness.

### Task 2: Trend Analysis

Multi year analysis of Adidas AG to identify changes in financial performance, assess business development, and support investment decisions.

### Task 3: Peer Group Analysis

Comparative analysis of BMW AG, Mercedes Benz Group AG, and Volkswagen AG using benchmarking techniques to evaluate competitive performance within the automotive industry.

## Tools Used

* Financial Ratio Analysis
* DAX40 Financial Data
