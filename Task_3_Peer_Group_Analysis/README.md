# Task 3: Peer Group Financial Analysis

## Objective

This project compares the financial performance of BMW AG against selected competitors using financial ratios and Tableau dashboards.

The objective was to determine the relative strengths and weaknesses of each company and develop an investment recommendation based on peer group benchmarking.

## Company Selected

BMW AG

BMW was analyzed alongside other major automotive companies to evaluate its competitive position within the industry.

## Analysis Performed

* Profitability Comparison
* Liquidity Comparison
* Solvency Comparison
* Efficiency Comparison
* Peer Group Benchmarking
* Tableau Dashboard Development
* Investment Recommendation

## Tools Used

* Tableau
* Excel Dataset
* Financial Ratio Analysis

## Key Learning

This project strengthened my ability to compare companies within the same industry, interpret competitive financial performance, and support investment decisions using benchmarking techniques.

## Real World Relevance

Investment analysts frequently compare companies against their competitors before making recommendations. Peer group analysis helps identify which companies outperform or underperform industry standards.

## Team Project

This analysis was completed as part of a group project during the Financial Analysis course at the University of Würzburg.
