# Task 2: Financial Trend Analysis

## Objective

This project evaluates the financial performance of Adidas AG over multiple years using trend analysis and Tableau visualizations.

The objective was to identify changes in key financial indicators over time and assess whether the company's financial position was improving or deteriorating.

## Company Selected

Adidas AG

Adidas was selected as a DAX40 company with sufficient historical financial data for trend analysis.

## Analysis Performed

* Revenue Trend Analysis
* Profitability Trend Analysis
* Liquidity Trend Analysis
* Solvency Trend Analysis
* Tableau Dashboard Development
* Investment Recommendation

## Tools Used

* Tableau
* Excel Dataset
* Financial Trend Analysis

## Key Learning

This project strengthened my ability to analyze financial performance across multiple reporting periods, identify patterns in company performance, and communicate insights through data visualization.

## Real World Relevance

Investors and financial analysts rarely make decisions based on a single year of data. Trend analysis helps identify whether a company's financial performance is consistently improving, declining, or remaining stable over time.

## Team Project

This analysis was completed as part of a group project during the Financial Analysis course at the University of Würzburg.
