# Task 1: Single Year Financial Analysis

## Objective

This project evaluates the financial health of Adidas AG using financial ratios calculated in Tableau.

The analysis focuses on one reporting period and examines the company's liquidity, profitability, solvency, and activity performance.

## Company Selected

Adidas AG

Adidas was selected because it is one of the largest sportswear companies in the world and a member of the DAX40 index, making it a suitable company for financial statement analysis.

## Analysis Performed

* Liquidity Analysis
* Profitability Analysis
* Solvency Analysis
* Activity Analysis
* Investment Recommendation

## Tools Used

* Tableau
* Excel Dataset
* Financial Ratio Analysis

## Key Learning

This project strengthened my ability to interpret financial statements, calculate financial ratios, build Tableau dashboards, and communicate investment recommendations using financial data.

## Team Project

This analysis was completed as part of a group project during the Financial Analysis course at the University of Würzburg.
